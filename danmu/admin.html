# 配置

在配置项里配置好 Admin 相关选项，第一次启动时会将密码写入数据库，后面删除配置文件中的密码即可，如果忘记密码，可以再将密码写入配置文件，使用新写入的密码登录管理后台。

此功能正在开发中，以后变动可能会比较大。
